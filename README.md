# Canteen_Sales_Dashoard

Canteen Sales Dashboard (Excel Project)

This project is a complete Excel-based data analysis dashboard created to study sales, profit, waste, and category performance in a college canteen.
It uses Pivot Tables, Pivot Charts, and Data Cleaning to convert raw data into meaningful insights.
